# Operation Vortex

A few days ago my mentor and I were reviewing a system we're building together.
I'm the mobile dev on the team. He started asking me questions about decisions
I'd made in the code — why I wrote it this way, what the implications were,
what it costs at scale, when it breaks. I didn't have clean answers. I had vibes.
That conversation made the gap between building something and understanding what
you built very clear. So this is the reset. 182 days, two roadmaps running
concurrently every single day: Systems Engineering + Python from absolute scratch,
and Flutter rebuilt from the ground up — even after a year of shipping real apps,
because it's different when you know the why. Every concept, every project, every
mistake documented publicly. This is the public contract.

## Tracks
- **Systems Engineering + Python** — the machine, the language, databases, APIs,

- **Flutter + Dart** — rebuilt from scratch with systems thinking at every layer
